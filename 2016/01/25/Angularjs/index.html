Angularjs

AngularJS 是的框架.
一个为动态WEB应用设计的结构框架(JavaScript)
有数据绑定 依赖注入可以省去很多繁琐的代码
使用不同的方法尝试去弥补HTML本身在构建应用方面的的不足
AngularJS通过使用我们称为标识符(directives)的结构，让浏览器能够识别新的语法。

例如：

  • 使用双大括号语法进行数据绑定
  • 支持表单验证
  • 能将逻辑代码关联到相关的DOM元素上
  • 能将HTML分组成可充用的组件
主要文件

angular.min.js
angular-route.js 配置路由 在index.html中引入
app.js 里面写自己配置的路由

index.html //需要引入app.js 以index.html为根路径

需要注意的几个点
  • 用url访问时需要/#/
  • 因为在index.html中配置的ng-app 所以index为入口

如何使用？

安装angular (三种方式)

  • 去官方网站下载
  • 使用bower安装 命令 bower install angular - -save-dev
  • 使用npm安装 命令 nam install angular - -save-dev

创建index.html

之类的元素即可。然后加上 ng-app
